# plant-shop

# Backend

```sh
cd back
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
# Back to plant-shop

npm run serve

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
